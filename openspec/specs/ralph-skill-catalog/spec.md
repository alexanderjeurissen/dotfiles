## Requirements
### Requirement: Deprecated ralph-setup skill is removed from repository
The repository SHALL not include the deprecated `ralph-setup` skill directory or its template artifacts after this change.

#### Scenario: Skill directory is absent
- **WHEN** repository skills are enumerated
- **THEN** `agents/skills/ralph-setup/` is not present

#### Scenario: Setup templates are absent
- **WHEN** repository template files are enumerated under the removed skill path
- **THEN** `PROMPT_plan.md`, `PROMPT_build.md`, `AGENTS.md`, and `IMPLEMENTATION_PLAN.md` templates from that skill are not present

### Requirement: Repository guidance does not prescribe dual-prompt Ralph setup
Repository guidance SHALL not direct users to a two-phase Ralph setup based on `PROMPT_plan.md` and `PROMPT_build.md` generated by `ralph-setup`.

#### Scenario: References to removed setup flow are cleaned
- **WHEN** users consult repository guidance related to Ralph setup
- **THEN** they do not encounter instructions that depend on `ralph-setup` or mandatory dual prompt generation

# Description Patterns

## Formula
Use:
"<What the skill does>. Use when <situations>. Keywords: <natural user terms>."

## Strong examples
- "Triage production incidents by collecting logs, metrics, and deploy signals, then proposing low-risk mitigations. Use when alerts fire, error rates spike, or users report outages. Keywords: outage, latency, 500s, rollback."
- "Creates spec-compliant Agent Skills with progressive disclosure and evaluation prompts. Use when creating or refactoring `SKILL.md` workflows. Keywords: skill, SKILL.md, frontmatter, references, validation."

## Weak examples
- "Helps with skills."
- "General assistant for many tasks."

## Rules
- Prefer specific verbs.
- Include trigger context.
- Include common user wording.
- Avoid broad, overlapping claims.
